=========================================
  プリセット取り込み用テンプレート
=========================================

このフォルダ構造をコピーして、各フォルダに画像を配置してください。
設定画面の「プリセットフォルダを選択」で、このフォルダを選択すると
全カテゴリの画像を一括で取り込めます。

【フォルダ構造】
preset-template/
├── dolls/          ... ドール画像（全身の人形）
├── backgrounds/    ... 背景画像
└── clothing/       ... 服画像
    ├── top/            ... トップス（Tシャツ、シャツなど）
    ├── bottom/         ... ボトムス（スカート、パンツなど）
    ├── dress/          ... ワンピース
    ├── shoes/          ... 靴
    ├── accessory/      ... アクセサリー（帽子、リボンなど）
    ├── underwear_top/  ... 下着（上）
    └── underwear_bottom/ ... 下着（下）

【対応画像形式】
- PNG（推奨：透過対応）
- JPG / JPEG
- GIF
- WebP

【画像の名前】
ファイル名がそのままアイテム名になります。
例: blue-tshirt.png → 「blue-tshirt」として登録

【ヒント】
- ドール画像は縦長（512x1024ピクセル程度）がおすすめ
- 服画像は透過PNGにすると綺麗に重なります
- 背景画像は横長がおすすめ
