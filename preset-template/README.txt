=========================================
  きせかえゲーム プリセット作成ガイド
=========================================

【フォルダ構造】
preset-template/
├── backgrounds/           ... 背景画像（全ドール共通）
│   └── *.png
│
└── doll-{ドールID}/       ... ドール別フォルダ
    ├── dolls/             ... ドール画像
    │   └── {ドール名}.png
    │
    └── clothing/          ... 服画像（このドール専用）
        ├── top/           ... トップス
        ├── bottom/        ... ボトムス
        ├── dress/         ... ワンピース
        ├── shoes/         ... くつ
        ├── accessory/     ... アクセサリー
        ├── hat/           ... ぼうし（フォルダ追加で自動認識）
        ├── socks/         ... くつした
        └── {任意}/        ... フォルダ名がカテゴリになる

【重要】
- ドールと服は同じプリセットでセット
- 同じドールポーズ・サイズで服を作成すること
- 背景は全ドール共通

【対応形式】
- PNG（推奨：透過対応）
- JPG / JPEG / GIF / WebP

【サイズ目安】
- ドール: 512x1024 ピクセル程度（縦長）
- 服: ドールと同じサイズ（重ねるため）
- 背景: 1920x1080 程度（横長）
